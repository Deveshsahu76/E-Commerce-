# SonicRaksha - MERN E-Commerce Platform

SonicRaksha is a full-stack MERN e-commerce web application built with a modern shopping UI, product discovery, cart management, checkout flow, authentication, order tracking, Razorpay-ready payments, and scalable backend APIs.

## Live Links

- Frontend: https://e-commerce-nu-bay.vercel.app/
- Backend API: https://e-commerce-backend-1i0x.onrender.com/api/products
- GitHub Repository: https://github.com/Deveshsahu76/E-Commerce-

## Project Overview

This project is designed as a production-style e-commerce platform where users can browse products, view product details, add items to cart, checkout, make payments, and track orders.

The application includes a React frontend deployed on Vercel and an Express/MongoDB backend deployed on Render.

## Features

### User Features

- Modern responsive homepage
- Product listing page
- Product search, category filter, and sorting
- Product details page
- Image gallery support
- Add to cart
- Quantity update and remove cart items
- Checkout page with address form
- Razorpay-ready online payment flow
- Cash on Delivery demo flow
- Login and registration
- User authentication with JWT
- My Orders page
- Order Details page
- Order cancellation support
- Responsive UI for desktop and mobile

### Backend Features

- REST API architecture
- JWT authentication
- Product APIs with pagination, search, filters, and sorting
- MongoDB indexes for scalable product queries
- Cart API with authenticated user cart
- Order API with stock reduction
- Razorpay order creation
- Razorpay payment signature verification
- Paid order creation after successful payment
- Order status and cancellation logic
- MongoDB Atlas database integration

## Tech Stack

### Frontend

- React.js
- React Router DOM

- React Hook Form
- React Toastify

### Backend

- Node.js
- Express.js
- MongoDB

- Razorpay

### Database
